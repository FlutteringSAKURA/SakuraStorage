    7. Kraken’s A.I. Programming

  [프로젝트 설명]
   
      <목적>  
      
         [다양한 종류의 인공지능 구현]
         
         - 길 찾기 인공지능 : A*방식, NavMesh 각각을 활용한 길 찾기 로직 구현
         - Clustering Process : 군집형태의 무리가 행동하는 양태를 현실적으로 구현
         - Crowd Simulation : 집단행동 인공지능 시뮬레이션 구현
         - BTs(Behavior Trees) : Root Node 및 Fuzzy Logic 기반의 행동 트리 구현 연습
         - Final Integration : 위의 다양한 인공지능 방식을 통합적으로 구현 연습
         
      <주요 내용>
      
          * 유한 상태 기계 Finite-State Machine(FSM) 
         
          1. 센서구현 (시각, 촉각) 
          
          2. 길 찾기 Pathfinding (기본 - Way Point방식의 정해진 길 따라가기) 
          
          3. A* pathfinding 
          
          4. NavMesh (네비게이션 메시) 
          
          5. 영역탐색 (Navigation Area - Layer) 
          
          6. 끊어진 영역 길찾기 - Auto Off Mesh Link(자동) / Manual Off Mesh Link(수동) 
          
          7. 군집처리 
          
          8. Flock - Rigidbody 미사용 / Rigidbody 사용 
          
          9. Simple Crowd Simulation - Using NavMesh 
          
          10. 행동 트리(BTs, Behavior Trees) 
          
          11. 퍼지 로직을 사용한 인공지능의 개선 
          
          12 Integration
