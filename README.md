  [프로젝트 명칭]
  
    3. Cyber Stadium – Screen Base Ball Project

  [프로젝트 설명]
   
      <목적>  
      
         현실적인 물리 엔진과 수학적 계산을 통한 현실적인 야구 시뮬레이션 및 시네머신을 활용한 화려한 볼거리 요소를 구현하여 전국의 스크린 야구장에 납품
         
      <주요 내용>
      
         1. Making Alpha version Pitcher Bot / Hitter Bot 
         2. Distance Calculator - 결과값에 대한 카메라 이벤트 연출을 위해서 타구의 거리를 예측 및 계산 
         3. 상황별 Cinemachine 연출 * 실내 스크린 야구 게임 프로토타입 샘플 프로젝트 
         4. 투수가 공을 던지고 타자가 공을 치는 경우를 Simulating하여 타격한 공의 Distance Value를 구하는 테스트. (파울 체크 포함) 
         5. 타자가 공을 타격했다는 가정하에 Predict Dis(예상 거리)와 Actual Dist(실제 거리)의 오차 범위. 
            즉 그 편차를 줄이기 테스트 - 이 오차범위 감소 테스트를 통하여 타격하는 순간 이미 어떤 이벤트를 발동시켜야 할지 미리 결정 지을 수 있는 이점을 얻고자 하였음.
         6. 투수 Animation과 Pitcher Bot의 Sync
